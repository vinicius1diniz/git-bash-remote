Comando Git Exemplo de uso Operação
config git config --global.user.name | user.email Configura o usuário e e-mail
help git help comando Obtem ajuda sobre um comando especifico
init git init Inicializar um repositorio em uma pasta
status git status Verificar a situaçao do repositorio
add (file) git add . | git add (arquivo) Adiciona um ou mais arquivos ao staging area
commit git commit -m "mensagem" Cria a versao(commita) o repositorio
reset git reset (arquivo) Volta do estado preparado para o modificado
restore git restore (arquivo) Desfaz uma alteração em um arquivo modificado
clean git clean -fd Desfaz a criação de um arquivo criado
branch git branch -M "main" Mudar o nome da branch principal
code code . | code (arquivo) Abre o VSCODE ou na pasta ou um arquivo
